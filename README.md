# Luact

Tiny React-like renderer experiments.

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start dev mode (build + local server + watch):

```bash
npm run dev
```

3. Open the URL shown by esbuild (usually `http://127.0.0.1:8000`).

## Build once

```bash
npm run build
```
